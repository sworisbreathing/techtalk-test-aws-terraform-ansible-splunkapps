Listers and stringreplace intentions

This example shows how to use stringreplace intentions instead of addterm intentions. Stringreplace is an alternative to addterm that is a bit more complicated and has some minor drawbacks. But it is useful if you want to do more than just add searchterms and key value pairs to the search.

Both panels contain a two-lister example that uses stringreplace intentions. The intention is created before the base search exists. This is counterintuitive but is perfectly acceptable and often very useful.
