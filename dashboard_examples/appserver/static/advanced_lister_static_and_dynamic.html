Static and dynamic listers

This view introduces static and dynamic listers.

The parent module in this view is a HiddenSearch module configured with the following search:

index=_internal source="*metrics.log" metrics

The parent module contains a StaticSelect plus ConvertToIntention pattern. The StaticSelect lister is configured explictly using module parameters.

Nested within these modules is a SearchSelectLister plus ConvertToIntention pattern. The SearchSelectLister contains a search that populates its selections.

The SubmitButton kicks off a search specified by the listers and a timerange picker. This search is passed on to two nested HiddenSearch modules.

The first HiddenSearch module modifies the search string and then pipes it to a timechart of sum(kb). It then displays result in a column chart, using HiddenchartFormatter.

The second HiddenSearch module does the same thing, but pipes it to a timechart to show the max(eps), min(eps) and avg(eps) over time, displaying the result in a line chart.
