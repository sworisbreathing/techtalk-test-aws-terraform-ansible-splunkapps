Advanced XML: Search performance 2

This is the second example on search performance. In this example we take away the FlashTimeline module and replace it with a HiddenFieldPicker module.

The fields of interest in the module are coded into the HiddenFieldPicker: host, index, source, sourcetype, clientip, status, and bytes. The user cannot change which fields to display.

However, searches in this view are about 10 times faster.

Technically, the searches are no longer submitted with required_field_list=*, but rather with the specified fields in HiddenFieldPicker.
