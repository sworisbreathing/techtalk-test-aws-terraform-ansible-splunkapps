A simple example: Three modules wired together

Below is an example of three nested modules:

HiddenSearch

TimeRangePicker
(layers its intentions on top of the HiddenSearch)

SimpleResultsTable
(kicks off the search and displays the results).

<!-- parent module -->
<module name="HiddenSearch" layoutPanel="panel_row2_col1" group="3 modules wired together" autoRun="True">
  <param name="search">index=_internal source=*metrics.log | chart sum(kb) by series | sort -sum(kb) | head 10</param>

  <!-- Child module adding the timerange on top of the base search -->
  <module name="TimeRangePicker">
    <param name="selected">Last 15 minutes</param>
    <param name="searchWhenChanged">True</param>

    <!-- This child module kicks off the search inherited from the parent modules -->
    <!-- the table automatically displays the first ten results from the search. -->
    <module name="SimpleResultsTable"></module>
  </module>
</module>
