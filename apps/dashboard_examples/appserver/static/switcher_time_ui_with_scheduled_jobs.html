Switcher with time controls

This example shows how to use a switcher to display a scheduled search for different time intervals.

The default module for the switcher automatically displays a column chart for a scheduled saved search. The scheduled saved search displays the last 15 minutes of results, so no time specification is needed for the default module. You can view details for the scheduled saved search by going to:

Manager > Searches and reports > Sample scheduled search

Subsequent modules explicitly run the same scheduled search, but specify the time range within their HiddenSearch modules. Here is the saved search run by all switcher modules.

index="_internal" source="*metrics.log" group="per_sourcetype_thruput"
| chart sum(kb) over series
