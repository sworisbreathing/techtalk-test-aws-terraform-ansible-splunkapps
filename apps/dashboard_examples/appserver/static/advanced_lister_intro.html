Advanced XML: Introduction to lister modules

Use lister modules to add lists to your dashboards. The examples on this page (and other lister pages) show various ways to use listers to link to other views. There is a wide range of possible configurations for listers. These are some of the more common ones.

Note: You can also use tables and charts to link to other views. This clickable behavior is referred to as drilldown behavior. Refer to the drilldown examples for details.

There are two types of lister modules:

  • Entity lister
    Builds lists from Splunk objects, such as users, saved searches, and others.

  • Search lister
    Builds list from searches run in a module.

The lister examples below show various ways to combine lister modules in a panel. The last lister example illustrates the complexity you can have when combining and nesting various modules in a panel.
