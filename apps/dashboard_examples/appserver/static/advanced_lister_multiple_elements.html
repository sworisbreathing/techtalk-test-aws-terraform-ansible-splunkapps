Lister driving multiple elements

This example shows how to use a lister to drive multiple elements. Highlights of this example:

  • Uses two separate copies of the <HiddenSearch plus HiddenChartFormatter plus JSChart> pattern.

  • Results displayed in timeline and in a SimpleResultsTable.

  • Uses GenericHeader modules to introduce the results.
