The other trick can be found in the Menu for this Dasboard Examples app.

The menu item Advanced XML > Static content and other tricks > Insert your own links navigates outside of this Dasboard Examples app to a separate web page.

This link is implemented directy from the nav menu, which you can find at:

$SPLUNK_HOME/etc/apps/dashboard_examples/default/data/ui/nav/default.xml

Here is the code snippet to implement the link: 

<collection label="Static content and other tricks">
  <view name="static_content"/>
  <view name="embedding_other_views"/>
  <view name="other_trick"/>
  <a href="http://google.com?q=yep you can put any external url into the nav">
    Insert your own links
  </a>
</collection>
