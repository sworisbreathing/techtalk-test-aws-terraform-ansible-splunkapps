Advanced XML: Advanced drilldown from charts

This view showcases drilldown patterns on JSChart modules.

It contains side-by-side panels of a JSChart and a SimpleResultsTable to help you compare and contrast these modules.

The source code for this view contains debugging features which are commented out. Go to the Manager page for this view to edit the source to try out the debugging. To access the actual source, go to:

Manager > Apps

For Splunk Dashboard Examples, select the following to open the Splunk XML Editor:

View objects > drilldown_chart1

Note: Changes you make in this editor are saved in the local directory. This prevents your customized code from being overwritten should you update the dashboard_examples app.

$SPLUNK_HOME/etc/apps/dashboard_examples/local/data/ui/views/drilldown_chart1
$SPLUNK_HOME/etc/apps/dashboard_examples/default/data/ui/views/drilldown_chart1
