Advanced XML: Search performance 1

This is the first of several examples on search performance. In this example we take away the FlashTimeline module, resulting in a modest increase in search performance. For comparison, the Lister with a form search example includes the FlashTimeline module.

Basically, this example illustrates that including the timeline does not bring on a performance hit.

Technically, the absence of the FlashTimeline module dispatches the search with status_buckets set to 1 instead of 300.
