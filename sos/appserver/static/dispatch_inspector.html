Dispatch Directory Inspector

This view provides insight into search artifacts stored in the dispatch directory, which you can find at the following location in your Splunk Enterprise installation:

$SPLUNK_HOME/var/run/splunk/dispatch/

When you run a search job, Splunk stores its results as an artifact in this directory, including information about the search such as:

  • The search string

  • The unique search ID

  • The TTL of the artifact (the time the artifacts remain on disk)

  • The user running the search job

  • The roles to which the user belongs

The list of current search artifacts is gathered using the rest search command against the search/jobs REST endpoint.

Important: The rest search command that is needed for this view is only available in Splunk Enterprise 4.3 or later versions. As a result, this view is not compatible with Splunk Enterprise 4.2.x and earlier versions.

For more information on search artifacts and the dispatch directory, refer to Manage jobs in the OS and Supervise your search jobs. Both links are in the Splunk Enterprise documentation.

Inventory of search artifacts

This panel lists all search artifacts found in the dispatch directory. Sort the table by different columns to gain insight on the contents of the dispatch directory. For example, sort on diskUsageMB to see which artifacts might have large disk space footprints. Or sort on TTL to see how long the search artifacts remain on disk.

Note: Typically, this view is most interesting for scheduled searches. Deselect Show scheduled searches only to include historical searches.

Breakdown of dispatch directory disk usage

This panel breaks down the search artifact information differently to provide additional insights into the contents of the dispatch directory. The search populating this panel aggregates statistics such as disk usage or artifact count by search name, user, or app. This allows you to rapidly find a common denominator for the heavier consumers of disk space in the dispatch directory.

Note: Typically, this view is most interesting for scheduled searches. Deselect Show scheduled searches only to include historical searches.

Searches used in this view

Click the View results link in each panel to open the search in the Search view. You can inspect, modify, and export the search results from the Search view.
