Crash Log Viewer

This view displays any splunkd crash logs recorded and dumped to: $SPLUNK_HOME/var/log/splunk/

If this view is empty, that means Splunk on Splunk did not find any crash logs.

Crash Log History

Provides a sortable list of all crash logs found.

The argv field indicates the nature of the crash: the main splunkd process or the search process.

Click on a crash log to populate the Crash Log Viewer and splunkd.log Viewer panels.

Crash Log Viewer

Displays the contents of a selected crash log.

splunkd.log Viewer

Displays messages logged in splunkd.log in the time context of the crash.

Messages are shown for the 5 minutes prior to the crash, with the most recent messages displayed first.

Searches used in this view

Click the View results link in each panel to open the search in the Search view. You can inspect, modify, and export the search results from the Search view.
