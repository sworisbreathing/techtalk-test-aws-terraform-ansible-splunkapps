Indexes Disk Usage and Properties

This view shows disk usage of indexes across a distributed deployment. It gathers its data from the services/data/indexes REST endpoint.

You can modify the results according to the following settings:

  • Show results for all indexers, or list results for a specific instance.

  • Show results for the top ten indexes, all indexes, or individual indexes.

  • List results by index, indexer, index by indexer.

  • Enable sorting by the metrics depicted in each panel.
    By default, sorting in each panel is by the metric value for that panel.

  • Enable a logarithmic scale to better view values that differ greatly in size.

  • In each panel, toggle between a chart and table to view results

Searches used in this view

Click the View results link in each panel to open the search in the Search view. You can inspect, modify, and export the search results from the Search view.
