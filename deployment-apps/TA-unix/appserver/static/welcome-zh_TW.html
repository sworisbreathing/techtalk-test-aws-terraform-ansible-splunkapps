Splunk *nix 應用是由輸入框, 組合視頁, 客製視頁, 以及搜尋指示所組成,
用於 Splunk 索引器, 或是 Splunk 傳收資料的設定, 能夠橫跨您的整體系統架構來收集系統量測記記錄, 系統記錄, 以及設定檔修改記錄等資料.

重要 - 所有 unix 應用使用的資料皆存於其專屬索引檔, 名為 "os". 在查詢 *nix 資料時, 請確認您的索引指令有加上 "index=os". 範例...

*nix 應用可以處理的檔案類型:

  • 一般記錄檔 (ex: /var/log)
  • 一般設定檔 (ex: /etc)
  • 檔案異動偵測 (by default /etc)
  • 若您想增加資料源, >> 請點我

*nix 應用也能夠處理一般系統指令所產出的結果:

  • top: 主機資源使用排名
  • vmstat: 記憶體使用狀況
  • iostat: 輸出入處理容量
  • ps: 處理序清單
  • netstat: 網路狀態及網路傳輸量
  • protocol: 網路傳輸量詳細資訊
  • interfaces: 乙太網路介面連接層狀態
  • open ports: 通訊埠開啟狀況
  • time: 系統時間
  • lsof: 檔案開啟清單
  
  • df: 磁碟機及磁碟區使用狀況
  • who: 上線使用者清單
  • users with privileges: 登入帳號清單
  • lastlog: 使用者最後登入時間表
  • cpu: 顯示 CPU 狀態
  • rlog: 以 ausearch 格式顯示系統記錄檔
  • packages: 已安裝的套裝軟體清單
  • hardware: 主機硬體資訊
