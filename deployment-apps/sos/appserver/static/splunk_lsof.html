File descriptor usage

Use this view to analyze the file descriptor usage of the main Splunk Enterprise processes: splunkd and Splunk Web. This view can help you find spikes or leaks (this occurs when a process is not freeing unused descriptors while continuing to create new ones) in file descriptor usage. The latest usage panels help you determine the type of usage, while the file descriptor usage over time panel helps you trend that usage over time and per process.

Note: At this time, this view is operative only on Linux and UNIX systems as it relies on events gathered by the lsof_sos.sh data input.

To display statistics for search peers, install the Splunk on Splunk Add-on on those instances.

  • S.o.S Add-on for UNIX and Linux

Enable the lsof_sos.sh scripted input

You must enable the lsof_sos.sh scripted input for this view to display any data. To enable this view, go to Manager > Data Inputs > Scripts. Look for the following data input and enable it:

$SPLUNK_HOME/etc/apps/sos/bin/lsof_sos.sh

After you enable the scripted input, S.o.S starts gathering resource usage information, sending it to the sos index.

Note: The data gathered by this input counts against the daily licensed volume.

Latest file descriptor usage for splunkd and Splunk Web

The two panels on the left provide the latest count and distribution of the file descriptors used by the splunkd process.

The two panels on the right display the same information for the Splunk Web process.

Mouse over slices in the pie charts to view additional information.

Note: The results displayed are from the last reading taken by the lsof_sos.sh script, which runs at 10 minute intervals. Typically, this reading is fairly current, but in some scenarios could represent older data.

File descriptor usage over time by process

The bottom panel tracks file descriptor usage over time. Use this panel to discover undesirable patterns such as spikes or leaks in file descriptor usage that might indicate problems.

Searches used in this view

Click the View results link in each panel to open the search in the Search view. You can inspect, modify, and export the search results from the Search view.
