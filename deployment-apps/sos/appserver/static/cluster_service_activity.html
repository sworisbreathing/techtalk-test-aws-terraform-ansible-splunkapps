Cluster Service Activity

Use this view to analyze the service job activity that is synchronizing the nodes of a cluster. Each panel is a time chart listing information about the replication service jobs. Use the dropdown menu to select a time period to analyze.

For an overview of Splunk Enterprise clusters and index replication, see to About clusters and index replication in the Splunk Enterprise documentation.

Size of bucket fix-up queues

A time chart that shows the sizes of the various bucket fix-up queues:

  • Data safety
    Buckets that do not have at least two minimum copies.

  • Generation
    Buckets that do not have a primary bucket assigned.

  • Replication factor
    Buckets that do not meet the replication factor.

  • Search factor
    Buckets that do not meet the search factor.

  • Streaming
    Buckets that are streaming.

  • Sync
    Buckets that are not synchronized between master and peers.

Service jobs activity

A time chart that shows the count of replication service jobs that have been added, that have been finished, and that are in the backlog to be completed.

Service jobs executed

A time chart that shows the count of replication service jobs.

Time spent per action

A time chart showing the time spent, in seconds, for the following actions of replication service jobs.

  • List the configuration

  • Create master buckets

  • Remove master buckets

  • Change the generation

  • List the generations

  • List buckets that are being serviced

  • List peer indexes

  • Create buckets on peers

  • Edit buckets on peers

  • List peers

  • List search heads

Size of service jobs queue

A time chart showing the number of service jobs in the queue.

Searches used in this view

Click the View results link in each panel to open the search in the Search view. You can inspect, modify, and export the search results from the Search view.
