Errors

Use the Errors view to determine which components of your Splunk Enterprise installation are reporting errors.

The Log File, Component, and Log Level pickers allow you to drill down to specific reporting events.

Use the Search Bar to further refine your query.

ERROR and WARN components panel

The top panel contains two charts that provide a snapshot of ERROR and WARN events reported in splunkd.log.

The number of events is scaled arithmetically to minimize the effect of large numbers.

ERROR and WARN events, by component
Depicts the number of events, by component, that report ERROR or WARN in splunkd.log

Total ERROR and WARN events, over time
Depicts the sum of ERROR and WARN events for each component, over time.

Drill down features panel

The bottom panel provides options to drill down for specific information:

  • Select specific log files, components, and log levels.
    Note: Log levels are NOT inclusive. For example, specifying WARN does not include INFO.

  • Specify a custom search on the Search Bar.
