Advanced XML: Search performance 4

This is the fourth example on search performance. This example does not use a field picker. However, that doesn't mean fields are absent entirely. Splunk still does summarization on:

  1. host, sourcetype, host, splunk_server

  2. Fields that the parser sees in the search commands themselves.

    For example if you have chart avg(bytes) over clientip Splunk collects sufficient statistics for bytes and clientip and those fields appear in the sidebar

Technically, the searches are no longer submitted with a required field list argument.
