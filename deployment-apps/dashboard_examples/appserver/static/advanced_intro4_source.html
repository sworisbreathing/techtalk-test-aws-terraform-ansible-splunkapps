Pattern examples

This example shows how to use the StaticContentSample module to add text to a panel.

By specifying the same value for row and column in the layoutPanlel attribute, the text specified in the StaticContentExample module appears in the HiddenSearch module.

  <module name="StaticContentSample" layoutPanel="panel_row2_col1">
    <param name="text">
      This example uses HiddenSearch, HiddenChartFormatter, and JSChart modules to display a bar chart created from a saved search.
    </param>
  </module>
  <module name="HiddenSearch" layoutPanel="panel_row2_col1" group="Charting the results of a saved search" autoRun="True">
