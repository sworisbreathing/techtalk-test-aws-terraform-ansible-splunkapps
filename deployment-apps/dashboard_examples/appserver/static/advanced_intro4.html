Advanced XML: Dashboard patterns

If you are curious to see complex real-world examples, take a look at the various dashboards that ship by default with Splunk's Search app. These dashboards use advanced XML (rather than simple XML).

Note: You can translate any simple XML view to advanced XML by adding the REST parameter, ?showsource=true to the view's URI.

The examples in this Splunk Dashboard Examples app slowly increase the level of complexity of dasboards used within Splunk's default views. Here are a few links to some dashboards in the Splunk Search app:

Pattern examples

This example shows how to use the StaticContentSample module to add text to a panel.

By specifying the same value for row and column in the layoutPanlel attribute, the text specified in the StaticContentExample module appears in the HiddenSearch module.

  <module name="StaticContentSample" layoutPanel="panel_row2_col1">
    <param name="text">
      This example uses HiddenSearch, HiddenChartFormatter, and JSChart modules to display a bar chart created from a saved search.
    </param>
  </module>
  <module name="HiddenSearch" layoutPanel="panel_row2_col1" group="Charting the results of a saved search" autoRun="True">
